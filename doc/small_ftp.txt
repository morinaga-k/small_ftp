*small_ftp.txt*	Plugin for using ftp on vim

Author: mori <y326045@yahoo.co.jp>
License: This file is placed in the public domain.

=================================================
*small_ftp* is a Vim plug-in to easily use the ftp.
You can use following commands.

 auto upload
 download file
 delete file
 rename file
 make directory

Maps:
	<Enter> -> get
	<del> -> del
	<Insert> -> rename
	<Home> -> mkdir

Command:
	:SmallFtp
